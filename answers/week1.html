TYDZIEŃ 1 - PODSUMOWANIE

Czym jest sekcja HEAD <head>, co powinno się w niej znaleźć?

  • Sekcja head odpowiada za metadane dokumentu html
  • Wewnątrz head mogą znaleźć się elementy: title, base, link, style, meta, script, noscript, template
  • link jest często używany do określenia pliku css, z którego ma skorzystać strona
  • meta to ogólny tag, reprezentujący dane, których nie da się przedstawić bardziej szczegółowym elementem

Dlaczego ustawiamy meta charset=UTF-8. Czy istnieją inne kodowania, czy można ich używać na stronie?

  • meta charset określa sposób kodowania znaków na stronie
  • Jedynym poprawnym kodowaniem stron w HTML5 jest UTF-8

Dlaczego ustawiamy title strony?

  • title odpowiada za tytuł strony, wyświetlany na zakładce
  • Tytuł strony może mieć duże znaczenie dla SEO

Co to jest sekcja body

  • Sekcja body zawiera treść strony, która jest widoczna dla użytkownika

Jak w html oznacza się najprostszy paragraf?

  • Za pomocą tagu p

Czym się różnią klasy od id?

  • id musi być unikalne dla całego deokumentu, jego zadaniem jest określenie pojedynczego elementu
  • class jest wspólne dla kilku elementów, które mają być tak samo traktowane przez CSS albo JavaScript

Na czym polega koncepcja przyjaznych linków (Friendly URL)

  • Przyjazne linki są łatwe w zrozumieniu dla każdego użytkownika
  • Są pozbawione szczegółów implementacyjnych strony czy aplikacji internetowej
  • Zawierają jedynie odniesienie do jakiegoś materiału, mogą przy tym odwzorowywać hierarchiczną strukturę strony/aplikacji
  • Są pozbawione elementów takich jak .html, ?id=1 i tym podobnych

Na czym polega model pudełkowy [CSS]?

  • W CSS, wszystko ma wokół siebie pudełko (box)
  • Box model (model pudełkowy) określa zachowanie rożnych części box: margin, padding, border i content.
  • Istnieją dwa sposoby określenia tego zachowania (są to opcje parametru box-sizing) - content-box, który jest domyślny, oraz border-box
  • Za pomocą display można ustawić opcje wyświetlania: inline lub block
  • block oznacza, że element zostanie przeniesiony do nowej linii, i wypełni całą dostępną szerokość (chyba że zostanie ustawiony parametr width). padding, margin i border spowodują wypchnięcie innych elementów poza box.
  • inline nie powoduje przeniesienia do nowej linii. W tym przypadku parametry height i width nie są stosowane. Niektóre elementy HTML stosują ten rodzaj wyświetlania domyślnie.
  • Więcej → https://developer.mozilla.org/en-US/docs/Learn/CSS/Building_blocks/The_box_model

Do czego służą nagłówki (h1,h2,h3) - dlaczego ważna jest struktura hierarchiczna nagłówków na stronie internetowej? Ile nagłówków można utworzyć maksymalnie na stronie?

  • Hierarchiczna struktura nagłówków ułatwia nawigację
  • Odpowiednio zagnieżdżone nagłówki przekazują sposób zorganizowania zawartości strony
  • Właściwa struktura nagłówków sprawia, że strona internetowa jest znacznie bardziej dostępna dla użytkowników czytników ekranu
  • Elementy nagłówkowe to tagi od <h1> - <h6>

Czy można zamieścić kilka nagłówków h1 na jednej podstronie?

  • Jest to zgodne ze standardem HTML, ale uważane za złą praktykę

Co robi tag <abbr>? A <ruby> istnieje <python>?

  • abbr to tag oznaczający skrócenie lub akronim. Jeśli zostanie zdefiniowany atrybut title, to po najechaniu na element abbr przeglądarka wyświetli zawartość title (powinno to być rozwinięcie skrótu)
  • tag ruby jest przeznaczony na małe annotacje, używany zazwyczaj do zaprezentowania wymowy azjatyckich znaków
  • tag python nie istnieje

Czym się różni tag <datalist> od <select>?

  • select pozwala jedynie na wybór jednej z podanych opcji, datalist umożliwia wpisanie własnej wartości
  • datalist może zostać związany z elementem input. Wtedy za pomocą atrybutu type na elemencie input można modyfikować sposób renderowania elementu datalist

Czym się różni tag <dl> od <table>, <ol> i <ul>?

  • dl to element opisujący listę definicji (określanych tagami dt oraz dd)
  • table reprezentuje tabelę
  • ol oraz ul przedstawiają listy, uporządkowaną i nieuporządkowaną

Do czego służy optgroup?

  • optgroup pozwala na zgrupowanie opcji wewnątrz elementu select

Czym się różni <q> od <blockquote>. Podaj wyraźny przykład użycia rozróżniający te dwa tagi

  • q służy do wyróżenienia cytatu w linii. blockquote reprezentuje dłuższy cytat, który rozpoczyna się od nowej linii i zazwyczaj jest przesunięty w prawo względem paragrafu

<sup> vs <sub>?

  • <sup> - indeks górny (an)
  • <sub> - indeks dolny (an)

Wyświetlane tak samo, mają różne funkcje: <pre>, <code>, <samp>, <kbd>.

  • pre reprezentuje tekst, który jest wyświetlany dokładnie tak jak zapisano (zachowane są spacje, łamanie wierszy itp)
  • code wskazuje tekst, który jest fragmentem kodu
  • samp reprezentuje dane wyjściowe programu komputerowego
  • kbd wskazuje na dane wejściowe, zazwyczaj z klawiatury

Co robi: input type=file?

  • Wyświetla przycisk pozwalający na wybór pliku z urządzenia

Jaki znacznik jest używany, by przełamać wiersz?

  • <br>

Za co odpowiada znacznik <hr>?

  • hr reprezentuje poziomą linię, która ma oddzielać różne sekcje dokumentu

Jak uzyskać efekt powiększenia pierwszej litery tekstu na określoną wysokość np. 4 wierszy?
